Merge the attributes from the ZCTA relationship file to the Oregon shapefile. Some ZCTAs span across counties. So we'll exclude the attributes that are county-specific; e.g., 2010 Population of the 2010 County (COPOP) and Total Area of the 2010 County (COAREA). The record file layout can be found here.

d1 <- shpOR@data
d2 <- subset(zctarelOR, select = c(ZCTA5CHR, ZCTA5, STATE, ZPOP, ZHU, ZAREA, 
    ZAREALAND))
d <- merge(d1, d2, by.x = "ZCTA5CE10", by.y = "ZCTA5")
shpOR@data <- d
summary(shpOR)

x <- shpOR@data$ZPOP

colcode <- findColours(class, pal)
